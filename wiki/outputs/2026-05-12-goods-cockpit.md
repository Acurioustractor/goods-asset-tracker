## How to use this Cockpit

The Cockpit is read-only in Notion. To update:

1. Edit the source markdown at `wiki/outputs/2026-05-12-goods-cockpit.md` in the Goods repo
2. Run `cd /Users/benknight/Code/act-global-infrastructure && node scripts/sync-goods-wiki-to-notion.mjs --cockpit`
3. The Cockpit page in Notion is replaced with the latest content
4. Subpages (each wiki output) are also synced if changed

If a wiki output is added or updated, sync it with: `node scripts/sync-goods-wiki-to-notion.mjs --file wiki/outputs/2026-05-12-new-file.md`

Edits made directly in Notion will be overwritten on next sync. The wiki repo is the source of truth.
