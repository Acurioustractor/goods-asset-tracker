## Reviewable Substance

The action dashboard is useful because it shows program follow-through. The mentor-facing version should only show the action themes, not the internal task mechanics.

| Action area | What it shows |
|---|---|
| Program compliance | Surveys, webinars, deadlines and QBE communication are being tracked. |
| Capital follow-up | SEFA, QBE match, LOIs and funder conversations have owners and next steps. |
| Procurement follow-up | Centre Corp, Miwatj, NPY, WHSAC/Groote and other buyer/partner signals can become tracked actions. |
| Diagnostic readiness | The ten QBE topic gaps can be turned into named tasks. |
| Governance | Actions can be reviewed before they become external commitments or CRM pushes. |

## Mentor-Safe Export

A weekly export should include:

- action theme,
- owner,
- status,
- due date,
- blocker,
- decision needed,
- related wiki/source page,
- whether it is internal, mentor-safe or external.

It should not include private contact notes, raw CRM data, household/story data or internal system implementation details.
