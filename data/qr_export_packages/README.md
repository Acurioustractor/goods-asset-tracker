## How to Use

1. **By Community**: Send each community folder to your local printer/coordinator
2. **By Product**: Useful if you want to print all beds or all washers together
3. **Manifest Files**: CSV files list all assets in each package for reference

## Printing Recommendations

- **Format**: SVG files are scalable - recommended for best quality
- **Size**: Print QR codes at least 50mm x 50mm for easy scanning
- **Material**: Use weatherproof/UV-resistant materials for outdoor assets
- **Testing**: Print a few test QR codes first and scan with phone to verify
